# Knife

A lightweight, single-user **ActivityPub** blog platform and social networking server built with Go.

## Documentation

Full documentation is available in the [docs/](docs/documentation.md) directory.

## Quick Start

1.  Build: `go build`
2.  Setup: `./knife setup`
3.  Run: `./knife`
4.  Open `http://localhost:8080`
